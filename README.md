# 온라인 클래스 툴즈
## 안녕하세요!
2020년 3월, 유례 없던 온라인 개학이 실시되었습니다. '온라인 클래스 툴즈'는 성실하게 온라인 클래스에 참여하고 있는 학생 여러분들을 돕기 위한 도구들/설명서들의 모임입니다.
## Disclaimer
면책 조항: 도구를 *올바르게* 사용하세요. 부정한 목적으로 사용해서 일어나는 어떠한 피해도 개발자/기여자는 책임지지 않습니다.
